# HiBerNET SSO Bridge FLFMOD的下载

- **类型**：`ssobridge`

## HiBerNET SSO Bridge — HMCL → Minecraft SSO Info Pass-through Mod

将 HiBerNET SSO 账户信息传递到 Minecraft 游戏内，支持系统参数和环境变量两种方式。支持各个实例独立管理。

### 使用方式
安装FLFHMCL项目，将此模组放入mods文件夹即可
