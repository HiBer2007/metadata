# HiBerNET SSO FLFMOD的下载

- **类型**：`flfHBNsso`

## HiBerNET-SSO-FLFHMCLmod
一个FLFHMCL模组，为HMCL拓展了对于HiBerNET SSO的OIDC登录支持
为了防止伪造客户端，此模组为闭源项目

### 使用方式
安装FLFHMCL项目，将此模组放入mods文件夹即可
